#### Coniugazione dei verbi

| ましょう              |             |         |
| ----------------- | ----------- | ------- |
| **forma gentile** | non-passato | passato |
| affermativo       | ます          | ました     |
| negativo          | ません         | ませんでした  |

| a               |                      |         |
| --------------- | -------------------- | ------- |
| **forma piana** | non-passato          | passato |
| affermativo     | forma del dizionario | ~た      |
| negativo        | ~ない                  | ~なかった   |

| IRU-ERU | CONSONANTE+U |
| ------- | ------------ |
| -る+ない   | -う+あない       |

来る diventa però こらい e non くらい  
する diventa però しない e non すない  

あう　(会う）-> う è un rimasuglio di "wu"(𛄟) e quindi あう diventa あわない  

ない non è altro che un aggettivo  
Per aggettivo si intende una parola che descrive, quindi verbo+ない è un aggettivo che indica l'assenza dell'azione  
Nel dizionario ない significa "inesistente" (食べない = assenza di mangiare), aggiungendo です si può fare la forma gentile  
Per questo in forma piana passata negativa ない diventa なかった  

In giapponese in una frase si possono fare 4 negazioni (doppia negazione è affermazione)  

Le negazioni secondo il Minna No Nihongo fino all'unità 19
